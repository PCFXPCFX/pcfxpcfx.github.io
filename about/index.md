# 博客功能与技术
本站基于 [Hexo](https://hexo.io/zh-cn/) 框架，使用 [AnZhiYu](https://github.com/anzhiyu-c/hexo-theme-anzhiyu) 主题，旨在为读者提供一个舒适、优雅的阅读环境。博客目前支持的功能大致如下：
- 暗色模式
- 中控台
- 音乐播放器
- LaTeX支持
- …
